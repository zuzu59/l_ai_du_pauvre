# l_ai_du_pauvre
Comment profiter du cloud de dev de Github pour faire tourner une intelligence artificielle ?


Installation

curl -fsSL https://ollama.com/install.sh | sh
ollama serve &


Utilisation

Mistral Nemo
ollama run mistral-nemo
